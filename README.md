# CARD CLASH - AR

Acesta este un nou repository creat din cauza unor probleme de integritate ale vechiului repository.

## Descriere

Am adăugat o **interfață funcțională de meniu** care suportă modul multiplayer.  
Am implementat **comunicarea în rețea între jucători**, inclusiv trimiterea de mesaje și sincronizarea stadiilor jocului.  

Logica jocului va fi preluată din vechiul repository și trebuie integrată în scena principală de joc (`Game Scene`) pentru a funcționa complet.
